# Automated Testing & Data Validation System (NHL Stats)

## Overview

End-to-end testing and data validation project using Java, Selenium, and SQL to verify UI elements and ensure data integrity using real-world NHL statistics.

---

## Features

* Automated UI testing with Selenium WebDriver
* Validation of player data (names, teams, stats)
* Pass/fail reporting with logging for debugging
* SQL-based data validation and anomaly detection
* ERD documentation (crow’s foot notation)

---

## Tech Stack

Java • Selenium • SQL • MySQL Workbench

---

## How to Run

1. Clone repo
2. Set GeckoDriver path
3. Run main class in IntelliJ/Eclipse
4. View console test results

---

## Sample Output

```
TEST: Player Data → PASS
TEST: Stat Validation → FAIL
ERROR: Invalid GP value
```
